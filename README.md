**Short Master Final Project — Clinical Decision Support on MIMIC-III**

A full end-to-end GenAI pipeline for clinical QA on unstructured ICU notes, designed for on-premises, compliance-aware deployment.

| Component | Details |
|---|---|
| **Retrieval** | Hybrid BM25 + S-PubMedBERT dense retrieval with RRF fusion |
| **Router** | Adaptive Query Router — dynamic sparse/dense switching |
